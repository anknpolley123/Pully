## 🛠️ Installation & Usage

### Prerequisites
Ensure your system has `git`, `php`, and `curl` installed.

### Setup
```bash
# Clone the repository
git clone https://github.com/anknpolley123/Pully.git

# Enter the directory
cd Pully

# Grant execution permissions
chmod +x pully.sh

# Run the tool
sudo bash pully.sh
